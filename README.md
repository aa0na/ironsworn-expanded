# Ironsworn Expanded

## About
---
This contains additional items from the Loadstar Expanded not included in Ironsworn or Delve.

## Features and Notes

* Ironsworn from [Shawn Tomkin](https://tomkinpress.com/pages/ironsworn)
* [Ironsworn: Loadstar - Expanded Reference Guide](https://tomkinpress.com/products/ironsworn-lodestar-expanded-reference-guide)
* Some Oracles/Rolltables are systen-agnostic, some require [Ironsworn & Starforged](https://foundryvtt.com/packages/foundry-ironsworn).

## Module Installation

To install the module, search for `Ironsworn Expanded` or `Ironsworn` in the `Add-On Modules` tab of of the Foundry VTT game setup screen. Then click on `Install`.

Or use this URL and click on `Install`:

```bash
https://github.com/aa0na/ironsworn-expanded/releases/latest/download/module.json
```

By default, the Ironsmith Oracles will be integrated into the Oracle Tree in the Character Sheet. The Ironsworn Classic and/or Ironsworn Delve rulesets must be enabled.
